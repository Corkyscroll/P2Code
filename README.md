#ATTRIBUTES
Remember Getters and Setters for all attributes.

Example:

private int x;

private int y;

public int getX()

{

  return x;

}

public int getY()

{

  return y;

}

public void setX(int x)

{

  this.x = x;

}

public void setY(int y)

{

  this.y = y;

}
